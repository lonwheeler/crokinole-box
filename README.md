# crokinole-box
VCarve Files for Crokinole Scoring Box V1.0

A simple drawing for Crokinole Scoring Box on VCarve to use on a Shopbot. Holds two sets of Player Buttons and Cribbage pegs for scoring.

Box Bottom
	Material
		      7"x10"x1.65" Oak
	
	Tools Required
			0.25" End Mill
			0.50" End Mill
	Cut Time 
		30 minutes
		
Box Top
	Material
		      6"x10"x0.84" Oak
		
	Tools
			.25" End Mill
			.125" End Mill
			V-Bit (60 deg 0.25")
	Cut Time 
		30 Minutes
